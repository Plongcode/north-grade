# North Garage

Современный сайт для детейлинг-центра, разработанный на Flask.

## Возможности

- Авторизация
- Панель администратора
- Онлайн-запись
- SQLite база данных
- Темная тема
- Два языка
- Адаптивный дизайн

## Технологии

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite

## Запуск проекта

```bash
pip install -r requirements.txt
python app.py
```

После запуска:

http://127.0.0.1:5000
